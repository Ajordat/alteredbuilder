This new release provides a way to tag your decks!

Tag your decks with one of the following playstyles: Aggro, Midrange or Control
And to further expand its categorization, add up to two of these tags: Combo, Disruption, Ramp and Token
It is also possible to search for decks using these tags, so make sure to categorize your creations so they are easily found!

  • {% translate "New features" %}
    • New system to tag decks according to their playstyle.
    • Functionality to search decks by their tags. Check the dropdown of filters on the decks tab!
    • Added ordering to the lists of decks. It's now possible to sort by:
      • Most loved decks
      • Most viewed decks
    • Added filtering to own decks.
    • Added infinite scrolling to own decks.
  • {% translate "Fixes" %}
    • Bug where the notification fetching failed for unauthenticated users instead of displaying a custom message.
    • Bug where client translations (e.g. popups and deck statistics) did not appear in the selected language.
  • {% translate "Other changes" %}
    • Added certain expensive queries to a cache to improve user experience. For example, the trending cards, the trending decks and the community statistics.
    • Limited the notification dropdown to the last 10 items.
    • Removed the limit of 20 rows when displaying the list of followers.
    • Minor UI improvements to deck list views.
    • Added minor highlight to changes when releasing a fix version (e.g. highlight the changes in this text when going from 1.11.0 to 1.11.1).
