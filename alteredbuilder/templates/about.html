About the project

Application to build, analyze and share Altered TCG decks.

This platform uses the following technologies:

  • Python 3.12 to implement the backend technology. Remarkable tools used are:
    • Black as the code formatter
    • Coverage.py as tool to measure the code coverage of the written tests
    • Django 5 as the web framework
      • Django Debug Toolbar in development mode as a troubleshooting and performance diagnosis tool
      • Django REST framework 3 to expose a limited part of the API
      • django-allauth to integrate a complete authentication mechanism that allows social accounts as well
    • Google API libraries to authenticate into Google and interact with the dependencies on GCP services
    • Unittest to construct and run unit tests
  • Google Cloud Platform (GCP) as the public cloud provider. Specifically, the products used are:
    • Artifact Registry: Storage for the service's docker images
    • Cloud Build: Implement the CI/CD pipeline. It builds the base image, pushes it into the registry, applies the database migrations, collects the static files and deploys the service.
    • Cloud Logging: Log management for the builds, the app and the database
    • Cloud Run: Serverless platform that runs the app and scales it according to the incoming traffic
    • Cloud SQL for PostgreSQL: A fully-managed database service with a PostgreSQL instance storing the service's database
    • Cloud Storage: Store the static files. Acting like a CDN
    • IAM: Authenticate and authorize access to resources within GCP; while adopting the least privilege security principle on Service Accounts
    • Secret Manager: Store and access for the application's secrets, such as the database credentials.
  • Docker as the containerization solution. In particular:
    • Docker Compose as the tool to spin up the multiple containers required while developing locally
    • Dockerfile to define the layers of the app's base image
  • HTML, CSS and JavaScript as the frontend technologies. With the help of:
    • Bootstrap 5.3: Frontend toolkit used as the base for most of the page elements
    • Font Awesome: Source for all the icons used
    • Google charts: Display of pie and bar charts
  • GitHub as the Version Control System
